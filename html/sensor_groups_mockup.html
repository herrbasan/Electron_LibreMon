Sensor Groups Configuration

Configure which hardware sensors LibreHardwareMonitor should poll. Disabling unused sensors reduces CPU usage and prevents unnecessary disk access.

Hardware Sensor Groups
Hardware Categories

Select which hardware types to monitor. Changes require restarting LibreHardwareMonitor (~3s interruption).

Usage Notes
  • CPU, GPU, Memory: Core monitoring sensors, typically always enabled
  • Motherboard: Voltage, fan speed sensors - enable if you need PSU monitoring
  • Storage: ⚠️ Polls HDDs every second - disable to reduce wear on mechanical drives
  • Network: Bandwidth monitoring - disable if you don't need network stats
