# PawnIO Installer

This folder should contain `PawnIO_setup.exe` from https://pawnio.eu/

The installer is run automatically during:
- First installation (`--squirrel-install`)
- Updates (`--squirrel-updated`)

## Download

1. Go to https://pawnio.eu/
2. Download the "Official edition" (signed)
3. Place `PawnIO_setup.exe` in this folder

## Minimum Version

LibreHardwareMonitor requires PawnIO 2.0.0.0 or later.

## Installation Arguments

The installer supports:
- `-install` - Silent install
- `-uninstall` - Silent uninstall
